# NextGoal Running - Public Pages

This repository hosts public pages for App Store Connect.

## Pages

- Privacy Policy: `index.html`
- Support: `support.html`

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. In the repository, open **Settings > Pages**.
3. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (root)
4. Save and wait for deployment.

Your site URL will be:

`https://<your-github-username>.github.io/nextgoal-privacy/`

So your App Store URLs should be:

- Privacy Policy URL: `https://<your-github-username>.github.io/nextgoal-privacy/index.html`
- Support URL: `https://<your-github-username>.github.io/nextgoal-privacy/support.html`
